# Keyword Strategy

How to research, organize, and assign keywords across every page type in the framework.

---

## 1. Keyword Research Process

### Step 1: Seed Keywords
Start with the core business descriptor. Ask: "What would my ideal customer type into Google when they need what I offer?"

```
Service + Location:     "plumbing services austin"
Service + Modifier:     "emergency plumber", "affordable plumber near me"
Problem-based:          "burst pipe repair", "leaky faucet fix"
Question-based:         "how much does a plumber cost"
```

### Step 2: Expand with Modifiers

Apply modifiers to the seed keyword to discover keyword clusters:

| Modifier Type | Examples |
|--------------|---------|
| Location | city name, neighborhood, "near me", zip code |
| Intent | "best", "affordable", "cheap", "professional", "certified" |
| Urgency | "emergency", "same-day", "24-hour", "fast" |
| Service stage | "installation", "repair", "replacement", "maintenance" |
| Audience | "residential", "commercial", "industrial" |

### Step 3: Classify by Search Intent

| Intent | Signal Words | Page Type |
|--------|-------------|-----------|
| Transactional | "hire", "book", "cost", "price", "near me" | Service, City, Landing |
| Commercial | "best", "top", "compare", "vs", "review" | Pillar, Comparison |
| Informational | "how to", "what is", "why", "guide" | Blog, FAQ |
| Navigational | Brand name, "contact", "phone number" | Homepage, Contact |

---

## 2. Keyword Mapping Matrix

Assign ONE primary keyword per page. No two pages compete for the same primary keyword.

| Page | Primary Keyword | Secondary Keywords |
|------|----------------|-------------------|
| Homepage | `{primary service} {city}` | `{brand}`, `{service} company`, `{service} near me` |
| Service Pillar | `{service name} {city}` | `{service} cost`, `{service} company`, `best {service}` |
| City Page | `{service} {city name}` | `{service} in {city}`, `{city} {service} company` |
| Keyword Page | `{long-tail keyword}` | Related variations |
| Blog Post | `{question keyword}` | Related questions, LSI terms |

---

## 3. Keyword Cannibalization Prevention

**Cannibalization** occurs when two pages target the same keyword and compete against each other. Prevent it by:

1. Maintaining a **keyword master sheet** (one row per page, one primary keyword per row)
2. Using the **canonical tag** on paginated or filtered views
3. **301 redirecting** duplicate pages to the strongest version
4. Writing **clearly differentiated content** — no two pages should cover the same angle

### Cannibalization Check

Before publishing a new page, search: `site:yourdomain.com "{keyword}"`

If more than one page returns, differentiate or consolidate.

---

## 4. Keyword Difficulty vs. Priority

Balance quick wins (low-difficulty, decent volume) with long-term targets (high-difficulty, high volume):

```
Phase 1 (Month 1–3):   Target KD < 30, volume > 100/mo  → Blog posts, FAQ, city pages
Phase 2 (Month 3–6):   Target KD 30–50, volume > 500/mo  → Service pillar pages
Phase 3 (Month 6–12):  Target KD 50–70, volume > 1k/mo   → Homepage, top-level service
Phase 4 (Year 2+):     Target KD > 70, high volume        → Authority / age builds this
```

## 5. City Page Keyword Template

For each city, generate one page targeting:

```
Primary:    "{primary service} {city}"
Secondary:
  - "{primary service} in {city}"
  - "{city} {primary service} company"
  - "{primary service} {city} {state}"
  - "{city} {primary service} near me"
```

---

## 6. Long-Tail Keyword Page Template

Long-tail pages target 3–5 word phrases with specific commercial or transactional intent:

```
Examples:
  - "emergency drain cleaning austin tx"
  - "water heater replacement cost austin"
  - "commercial plumbing contractors near austin"
  - "same day plumber south austin"
```

Each long-tail page:
- Is a minimum 600 words
- Targets one exact long-tail phrase
- Links to its parent pillar page
- Links to 2–3 related long-tail pages

---

## 7. Blog Content Keyword Strategy

Blog posts target informational keywords to build topical authority and capture top-of-funnel traffic:

```
Question format:   "how to unclog a drain without chemicals"
"Best X" format:   "best water heaters for homes in austin"
Comparison:        "tankless vs tank water heater: pros and cons"
How-to guide:      "how to prevent frozen pipes in winter"
Local + info:      "average plumbing costs in austin 2024"
```

**Every blog post must link to at least one service pillar page** with a contextual CTA.

---

## 8. Keyword Tracking

Track keyword positions weekly. Flag any keyword that:
- Drops more than 5 positions week-over-week → investigate and update page
- Reaches position 4–10 → optimize for click-through rate (title + description)
- Reaches position 1–3 → protect with link building and content freshness updates
